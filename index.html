Instalar AppArmor en Archlinux

vm_jorge publicado el 2024-05-08 incluido en Linux  Archlinux
📶 Nivel de dificultad MEDIO ¿Que es AppArmor y porque lo necesito? Definicion de Wikipedia: AppArmor (“Application Armor”) es un módulo de seguridad del kernel Linux que permite al administrador del sistema restringir las capacidades de un programa. Para definir las restricciones asocia a cada programa un perfil de seguridad. ¿Pero para que lo necesitaria yo, si no soy Sysadmin? AppArmor ayuda a prevenir que las aplicaciones accedan a recursos del sistema que no necesitan para su funcionamiento normal, lo cual puede limitar el daño potencial en caso de que una aplicación se vea comprometida por un ataque malicioso.
